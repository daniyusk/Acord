## About

Acord is a fork of [Dorion](https://github.com/SpikeHD/Dorion), created to evolve the lightweight Discord client experience with its own identity, a modern interface, and strong performance.

The project remains compatible with upstream components where needed. Therefore, some internal names and URLs retain `Dorion`, such as `Dorion Helpers` and `https://spikehd.dev/shelter-plugins/dorion-helpers/`. They identify plugins from the original ecosystem, not the Acord brand.

## Status

Acord is in early development. Builds and releases are not available yet.

## Goals

- A more modern and responsive Discord experience.
- Low resource usage and strong performance.
- Compatibility with themes, plugins, and useful components from the Dorion/Shelter ecosystem.

## Security

Client mods, plugins, and themes are disabled by default. See the [security model](docs/security.md) before enabling them.
